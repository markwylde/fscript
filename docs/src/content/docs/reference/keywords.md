# Keywords

Reserved keywords in Draft 0.1:

- `import`
- `from`
- `export`
- `type`
- `if`
- `else`
- `match`
- `try`
- `catch`
- `throw`
- `defer`
- `yield`
- `true`
- `false`

Reserved built-in type names:

- `Number`
- `String`
- `Boolean`
- `Null`
- `Undefined`
- `Never`
- `Unknown`

## Practical note

These names are reserved by the grammar and should not be used as ordinary identifiers.
