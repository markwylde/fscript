---
title: Your First Program
description: Write a small .fs file, check it, and run it through the current FScript CLI.
---

Create a file called `hello.fs`:

```fs
'hello world'
```

## Check it

```sh
fscript check hello.fs
```

## Run it

```sh
fscript run hello.fs
```

You can also use the shipped example:

```sh
fscript run examples/hello_world.fs
```
